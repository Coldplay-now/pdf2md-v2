# 项目图片资源

## 📸 截图说明

请将项目界面截图保存为 `screenshot.png` 放在此目录下。

### 推荐规格

- **文件名**: `screenshot.png`
- **格式**: PNG
- **尺寸**: 建议宽度 1200-1600px
- **内容**: 项目主界面展示

### 当前需要的截图

- `screenshot.png` - 项目主界面（上传区域、进度显示等）

## 📋 截图内容建议

好的项目截图应该包含：
- ✅ 清晰的界面展示
- ✅ 主要功能可见
- ✅ 美观的配色
- ✅ 真实的使用场景

## 🎨 其他图片资源（可选）

如需添加更多展示图片，建议命名规范：
- `feature-upload.png` - 上传功能展示
- `feature-progress.png` - 进度展示
- `feature-result.png` - 结果展示
- `demo.gif` - 动态演示
