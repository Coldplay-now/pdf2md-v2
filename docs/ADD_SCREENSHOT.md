# 📸 添加项目截图说明

## 需要添加截图

README中引用了项目截图，需要将界面截图添加到此位置：

### 📁 文件路径
```
docs/images/screenshot.png
```

### 🎯 截图内容

请截取项目主界面的截图，包含：
- 📤 上传区域（拖拽上传框）
- 📊 进度显示区域
- 🎨 整体界面设计

### 📝 操作步骤

1. 启动项目：`cd pdf-to-md-converter && bash start.sh`
2. 访问：http://localhost:8000
3. 截取整个页面或主要区域
4. 保存为 PNG 格式
5. 重命名为 `screenshot.png`
6. 放到 `docs/images/` 目录
7. 提交：`git add docs/images/screenshot.png && git commit -m "添加项目截图"`

### 💡 提示

- 推荐使用浏览器全屏模式截图
- 确保界面清晰美观
- 可以先上传一个测试文件，展示正在处理的状态会更有吸引力
